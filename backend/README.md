## Description

YemenCareers backend with [Nest](https://github.com/nestjs/nest). See also the [APIs](./APIs.md) documentation.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
